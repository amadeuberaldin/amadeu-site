Livro de Receitas — Chimias Artesanais

Aplicativo desenvolvido em Flutter para apoiar o cálculo e o preparo de chimias artesanais, usando proporções por peso, informações nutricionais e uma interface pensada para tornar o processo mais claro e organizado.

Ler visão geral Ver GitHub

Resumo

  • App em Flutter para cálculo e apoio ao preparo de receitas artesanais.
  • Uso de proporções por peso da polpa para calcular açúcar automaticamente.
  • Projeto usado como estudo de layout, navegação e organização visual em mobile.

Visão geral

Um app simples no escopo, mas valioso como exercício real de produto e interface.

Este projeto foi criado para transformar receitas artesanais em uma experiência de uso mais organizada e prática. Em vez de tratar o app apenas como vitrine visual, a proposta foi estruturar uma ferramenta funcional, capaz de apoiar cálculo, consulta e preparo de receitas com interface própria e foco em clareza.

Objetivo

Desenvolver um aplicativo mobile funcional que combinasse cálculo, organização de conteúdo e apresentação visual consistente, servindo também como estudo prático de Flutter.

Abordagem

Organizar o app em torno de receitas reais, cálculos baseados em peso e telas visualmente coerentes, priorizando usabilidade e legibilidade.

Estado

Projeto funcional para estudo de Flutter, organização de layout e preparação para publicação futura na Google Play Store.

O que o aplicativo inclui

Cálculo automático

O app calcula a quantidade de açúcar com base no peso da polpa, reduzindo erro manual e facilitando o preparo.

Informação nutricional

Cada receita traz referência nutricional por 100g e também por porção individual de 5g.

Processo detalhado

O preparo é apresentado de forma organizada, tornando o app útil não só como cálculo, mas também como guia de execução.

Identidade visual própria

O projeto foi usado para estudar composição visual, coerência entre telas e uma linguagem de interface mais cuidadosa.

Tipo de projeto

Mobile Flutter Estudo aplicado

Mais do que um protótipo visual, este projeto funciona como exercício prático de estrutura de app, navegação, apresentação de dados e consistência de interface em ambiente mobile.

O valor deste projeto está em transformar uma necessidade concreta em uma interface funcional, clara e organizada, enquanto consolida fundamentos de Flutter no processo.

Aprendizados e decisões de implementação

Flutter como ambiente de prática

O projeto serviu para praticar estruturação de telas, composição de widgets e organização de fluxo em uma aplicação mobile real.

Interface orientada a clareza

As telas foram pensadas para facilitar leitura, navegação e consulta rápida, em vez de depender apenas de apelo visual.

Conteúdo útil, não só demonstração

O app foi estruturado em torno de receitas reais e proporções práticas, o que dá mais sentido ao projeto do que um CRUD genérico de estudo.

Base para publicação futura

Além do aprendizado técnico, o projeto foi pensado como base para uma publicação futura na Google Play Store.
